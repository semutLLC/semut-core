# Semut Core Vision

Semut Core exists to make Semut applications reusable, consistent, and extensible.

The long-term vision is one AI assistant that can coordinate many parts of life without rebuilding the same infrastructure for every domain.

Instead of building every app from scratch, Semut verticals should share the same foundation.

## Vision Statement

Semut Core is the coordination layer for fragmented human activity.

It allows different Semut applications to reuse the same concepts:

* traces
* workspaces
* memory
* preferences
* policies
* calendars
* notifications
* integrations
* AI-assisted actions

## Why It Matters

A tennis app, ski app, business assistant, and health assistant may look different on the surface.

But underneath, they share the same coordination pattern:

```text
User goal
↓
Fragmented systems
↓
Captured traces
↓
Structured memory
↓
Policy-guided action
```

Semut Core captures that shared foundation.

## Long-Term Goal

A future developer or AI coding agent should be able to create a new Semut vertical by reusing Semut Core instead of starting from zero.

Example:

```text
semut-tennis
semut-ski
semut-business
```

Each vertical should feel like part of the same Semut ecosystem.
